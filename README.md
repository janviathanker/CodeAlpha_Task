# CodeAlpha_Iris_Flower_Classification
This repository is for the course DataScience. Iris Flower Classification project developed as part of CodeAlpha Internship. This project uses machine learning (KNN algorithm) to classify iris species based on flower measurements, showcasing data preprocessing, model building, and performance evaluation.
# 🌸 Iris Flower Classification (CodeAlpha Project)

## 📌 Overview
This project focuses on classifying Iris flower species using machine learning. Based on flower measurements such as sepal and petal dimensions, the model predicts whether the flower belongs to Setosa, Versicolor, or Virginica.

This project was developed as part of the CodeAlpha Internship to demonstrate fundamental concepts of supervised learning and classification.

---

## 🎯 Objective
- Build a machine learning model to classify Iris flowers  
- Understand classification algorithms (KNN)  
- Evaluate model performance using standard metrics  

---

## 📊 Dataset
The dataset used is **Iris.csv**, which contains:
- Sepal Length  
- Sepal Width  
- Petal Length  
- Petal Width  
- Species (Target Variable)  

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 🧠 Machine Learning Approach
- Data Preprocessing  
- Label Encoding  
- Train-Test Split  
- Model Training using K-Nearest Neighbors (KNN)  
- Model Evaluation  

---

## 💻 Implementation Steps
1. Load dataset (Iris.csv)  
2. Remove unnecessary columns (Id)  
3. Encode categorical target values  
4. Split data into training and testing sets  
5. Train KNN model  
6. Make predictions  
7. Evaluate performance  

---

## 📈 Results
- Achieved ~100% accuracy on test data  
- Model performs very well due to well-separated classes  
- Minimal classification errors  

---

## 📊 Evaluation Metrics
- Accuracy Score  
- Confusion Matrix  
- Classification Report (Precision, Recall, F1-score)  

---

## 🔍 Key Insights
- Petal length and width are the most important features  
- Setosa species is easiest to classify  
- KNN performs effectively on small datasets
